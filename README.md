# 🐍 PyPanel Ultra: Infinity

**PyPanel Ultra** は、ブラウザ上で動作する「プロジェクト指向」の統合開発環境です。
ファイル間の連携（importやlink）を自動的に解析し、すべてのファイルをまとめて実行します。

---

## 🔧 復活・強化された機能

1. **プロジェクト同時実行**
   - **Web**: `index.html` 内の `<link>` や `<script>` タグを解析し、サイドバーにあるファイルを自動的に埋め込んで実行します。
   - **Python**: 仮想ファイルシステム内の全ファイルをワーカーに転送するため、`import my_module` のように別ファイルの関数を呼び出せます。

2. **エディタ機能 (Monaco)**
   - **全角スペース検知**: 日本語の空白（　）をオレンジ色で可視化。
   - **ミニマップ**: 右側のマップをクリックしてジャンプ可能。
   - **コピー許可**: ターミナルの出力を選択・コピー可能に修正。

3. **スマホ対応**
   - **タッチリサイズ**: 画面下のバーを指でドラッグして、ターミナルの高さを調整可能。
   - **サイドバー**: モバイルでもスムーズに開閉可能。

---

## 📂 使い方

1. **ファイル作成**: 
   - `src/main.py` や `css/style.css` のようにスラッシュ区切りで入力すると、フォルダ階層が作られます。
2. **実行**: 
   - どのファイルを開いていても `▶ Run` を押せば、メインファイル（`index.html` や `main.py`）を起点にプロジェクト全体が実行されます。
3. **シェル**:
   - 下のターミナルで `ls`, `mkdir`, `rm` などのコマンドが使えます。
