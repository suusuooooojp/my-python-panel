# 🐍 PyPanel Ultra (Command Terminal Edition)

**PyPanel Ultra** は、ブラウザ上で動作する高機能な統合開発環境（IDE）です。  
ファイルシステム（ディレクトリ階層）やターミナルコマンドをサポートし、まるでローカルPCのような開発体験を提供します。

---

## ✨ 新機能

- **💻 ターミナル & コマンド**  
  画面下のターミナルでコマンド入力が可能になりました。
  - `ls`: ファイル一覧表示
  - `cd`: ディレクトリ移動
  - `mkdir`: フォルダ作成
  - `touch`: ファイル作成
  - `rm`: 削除
  - `python <file>`: Pythonスクリプト実行
  - `cat <file>`: ファイル内容を表示

- **📂 階層構造（入れ子）**  
  フォルダを作成し、ファイルを整理できます（例: `src/utils/math.py`）。
  エクスプローラーはツリー表示に対応しました。

- **🔧 自動言語判定**  
  言語選択バーを廃止。ファイルの拡張子（`.py`, `.js`, `.java` 等）から自動的に言語モードと実行ランタイムを切り替えます。

- **🧹 軽量スタート**  
  初期状態では Java や Go などの重いファイルは含まれません。必要な場合のみ拡張子を指定して作成してください。

---

## 🚀 インストール

1. 以下のファイルをWebサーバーに配置します。
   - `index.html`
   - `main.js`
   - `py-worker.js`
   - `sw.js`
2. ブラウザで `index.html` にアクセスします。
